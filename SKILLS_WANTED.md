- Rust
- Golang
- Stripe
- Vue
- Svelte
- Ruby on Rails
- Django
- Prisma
- tRPC
- GraphQL
- Kubernetes
- Terraform
- AWS CDK

If you're interested in contributing any of these skills, please open a Skill Request issue or submit a Pull Request.
